What is Roll20?

Roll20 is a free to use online virtual table-top environment implemneted through a browser based web app. Roll20 offers a variety of free features such as a 3D dice roller; voice, video, and text chat; character tracking; and more. For more detailed information reguarding Roll20, go to the Roll20 wiki using the link below.

How will Roll20 be used?

We will be using Roll20's virtual table-top for online sessions, sepecificaly for running combat, handling dice rolls, and for its built-in video chat function. We will, however, not be using the built in audio chat feature as we will be using other software for this task. You can access the Roll20 campaign page through the Roll20 cite or can visit the page directly using the link bellow.

How do I create an acount?

Creating an acount is simple, simply go to the main Roll20 website using google or the link below and click on Create Free Acount. Follow the instructions provided and you should be good to go.
